Inspirationen

Konzeptioneller Ansatz

Die Seite soll unlesbar sein, je breiter der Browser ist. Zusammengeschoben ergibt sich das Albumcover und der Songtext von El Buho von Blanco White.

Gestaltungsmittel / Technische Umsetzung

Je breiter der Browser ist, desto mehr ziehen sich die Buchstaben / Bilder / Texte auseinander.

Javascript, das jeden Buchstaben in ein div packt, und dieses mittels left, top im wrapper platziert, je nach dem wie breit das Browserfenster ist, wird das div von seiner Ursprungsposition entfernt.

Der Entwicklungsprozess

Window addEventListener() ruft browser_Window_Resized() Funktion auf. Diese verschiebt/skaliert jedes einzelne Element des DOMs, je nachdem wie breit der Browser ist.
