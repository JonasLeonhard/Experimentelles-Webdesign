Inspirationen

Konzeptioneller Ansatz

Bei der responsiven Vielfalt sollen verschiedene Gestaltungsansätze bei verschiedener Browserweite angezeigt werden.

Gestaltungsmittel / Technische Umsetzung

Mittels Mediaqueries werden verschiede Styles auf verschiedene Breiten angewandt.

Javascript ersetzt die Bilder bei verschiedenen Breiten und erzeugt einen Typing effect mit verschiedenen Zitaten aus einem Array (quotes).

Der Entwicklungsprozess

Mittels Photoshop erzeuge ich verschiedene SVGs, die ich als Hintergrund für die verschiedenen Views setze. Darüber platziere ich die restlichen Elemente in einem Grid.

Ich erstelle ein Script, welches aus einem quotes Array die verschiedenen Zitate in das DOM einsetzt.
